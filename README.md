# Healthcare-Analysis

Project Overview

This project combines SQL-based healthcare analysis with a Power BI executive dashboard to analyze patient admissions, hospital operations, billing, insurance revenue, and diagnostic test outcomes. The project is structured to reflect real-world healthcare reporting and analytics use cases.


---

Tools & Technologies

SQL: Data extraction, aggregations, window functions, ranking, and growth analysis

Power BI: Data modeling, DAX measures, KPIs, and interactive dashboards
